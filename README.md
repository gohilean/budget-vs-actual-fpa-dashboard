# budget-vs-actual-fpa-dashboard
Interactive Power BI FP&amp;A dashboard analyzing budget vs. actual performance, variance analysis, KPIs, and executive financial reporting.

This project simulates a real-world Financial Planning & Analysis (FP&A) dashboard built in Microsoft Power BI. The dashboard compares budgeted financial performance against actual results across multiple departments, allowing users to identify variances, monitor KPIs, and evaluate monthly financial performance.

---

## Business Problem

Finance teams regularly compare budgeted performance with actual results to understand whether departments are meeting financial expectations. This dashboard provides an executive-level view of financial performance while enabling deeper variance analysis by department and month.

## Key Business Insights

- Compared over **$28 million** in budgeted and actual financial performance.
- Identified department-level budget overruns and favorable variances.
- Highlighted monthly performance trends using interactive Power BI visuals.
- Built executive-level KPIs to quickly assess organizational financial health.
- Enabled drill-down analysis for financial decision making.
